# Minimal Unix Blog Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Convert the generated Hexo blog into a plain early-web Unix document archive.

**Architecture:** Keep the checked-in generated static site. Implement the visual change with one appended override block in `css/main.css`, preserving the existing Fluid HTML structure and the current dirty JavaScript changes. Use CSS to relabel mojibake navigation text, collapse the banner, convert homepage cards into date/title rows, and make post pages read like plain documents.

**Tech Stack:** Static HTML generated by Hexo 7.3.0, Fluid theme output, Bootstrap 4 classes already in the generated HTML, plain CSS, existing vanilla/jQuery theme scripts.

In `js/boot.js`, ensure `Fluid.boot.registerEvents` contains this call after `Fluid.events.registerScrollTopArrowEvent();`:

```js

```

In `js/events.js`, add this method after `registerScrollTopArrowEvent` and before `registerImageLoadedEvent`:

```js
  registerPostBackEvent: function() {
    var post = document.querySelector('.post-content');
    if (!post || document.querySelector('.post-back-link')) {
      return;
    }

    var back = document.createElement('a');
    back.className = 'post-back-link';
    back.href = '/';
    back.setAttribute('aria-label', '返回');
    back.innerHTML = '<i class="iconfont icon-arrowleft" aria-hidden="true"></i><span>返回</span>';
    back.addEventListener('click', function(event) {
      if (!document.referrer) {
        return;
      }

      try {
        var referrer = new URL(document.referrer);
        if (referrer.origin === window.location.origin && referrer.href !== window.location.href) {
          event.preventDefault();
          window.history.back();
        }
      } catch (error) {
        // Fall back to the home link if the referrer cannot be parsed.
      }
    });

    post.insertBefore(back, post.firstChild);
  },
```
